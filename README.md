🎯 Как использовать:
Запустите локально llama-server 127.0.0.1:8080 с вашими параметрами (убедитесь, что --mmproj указан)
Запустите приложение: python main.py
Добавьте изображение:
📷 Кнопка «Загрузить фото» → выберите файл
📋 Кнопка «Вставить из буфера» или Ctrl+V → вставьте скриншот
(Опционально) Добавьте подсказки в поле «Дополнительные указания»
Нажмите «🧠 Получить решение» → модель распознает текст и формулы, затем даст пошаговое объяснение
🧮 SymPy проверка → если в ответе есть формулы $$...$$, можно проверить вычисления
